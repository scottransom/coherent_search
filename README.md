This is a test project to make a python numpy version of fast, complex
Fourier interpolation, and to use that to perform coherent harmonic summing pulsation searches of PRESTO-style FFTs.

References:
  - Fourier interpolation is defined in Eqn. 30 of this paper:
    https://arxiv.org/pdf/astro-ph/0204349
  - PRESTO is the pulsar analysis toolset found here:
    https://github.com/scottransom/presto
